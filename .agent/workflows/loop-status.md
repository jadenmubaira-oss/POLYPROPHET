# Loop Status

- active objective
- current phase
- last completed verification gate
- open blockers
- whether `README.md` reflects the latest repo truth
- whether the loop should continue, pause, or stop

## POLYPROPHET Focus

Always surface any gap between:

- local code state
- README / AGENTS handoff state
- live runtime proof state

Do not describe the project as fully autonomous or fully live-proven unless a real accepted live order has been verified.
