# Supreme Deity Cloud Oracle

24/7 Cryptocurrency Prediction Engine

## Deploy to Render (Free)

1. Click "New" → "Web Service" on render.com
2. Connect this GitHub repo
3. Use these settings:
   - Build Command: `npm install`
   - Start Command: `npm start`
4. Click Deploy!

## Keep Alive (Free 24/7)

After deploying, go to uptimerobot.com and create a free monitor:
- Type: HTTP(s)
- URL: Your Render URL
- Interval: 5 minutes
